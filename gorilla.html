How We Made It

My friend Luke and I made this game using the Unity Game Engine. We wrote all of the game scripts in C#, and created all of the art from scratch using Adobe Illustrator. We worked on this project in our free time, and had a lot of fun making it. Luke lives in Louisiana, so we used skype for communication, and github for version control and collaboration.

The Game

The game is a 2d, infinite sidescroller. The game mechanics were inspired by Robot Unicorn Attack. Get a high score by collecting items, and surviving for as long as possible.
